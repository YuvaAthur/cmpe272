## Built With
* [Flask] (https://www.palletsprojects.com/) - The web framework used

## Deployment

```
* (venv) $ export FLASK_APP=microblog.py
* (venv) $ sudo bash
* (venv) root $ . venv/bin/activate
* (venv) root $ flask run --host=0.0.0.0 --port=80
```
